# AI Resume Screening System Deployment on Azure using Terraform

## Overview

This project demonstrates the deployment of a Java Spring Boot based AI Resume Screening System on Microsoft Azure using Terraform.

The infrastructure is provisioned as code (IaC) and includes networking, security, virtual machine deployment, PostgreSQL configuration, and application deployment.

---

## Technologies Used

* Terraform
* Microsoft Azure
* Ubuntu Linux
* Java 17 / Spring Boot
* PostgreSQL
* Swagger/OpenAPI
* Git & GitHub

---

## Infrastructure Provisioned

* Resource Group
* Virtual Network
* Subnet
* Public IP
* Network Interface
* Network Security Group
* Linux Virtual Machine
